# ExceptionHandlerAndSerilog.WebAPI

ASP.NET Core Web API üzerinde **Serilog**, **Global Exception Handling** ve **.NET Aspire** entegrasyonu denemeleri içeren bir eğitim projesidir.

## Özellikler

- Merkezi hata yönetimi için `IExceptionHandler` kullanımı
- Serilog ile console loglama
- Development ortamında OpenAPI desteği
- Scalar API arayüzü
- .NET Aspire service defaults entegrasyonu

## Kullanılan Teknolojiler

- .NET 10
- ASP.NET Core Web API
- Serilog.AspNetCore
- Microsoft.AspNetCore.OpenApi
- Scalar.AspNetCore
- .NET Aspire
